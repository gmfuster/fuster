Probabilities are expressed between 0 and 1. (O = not happening and 1= for sure)

In the image the possible outcomes are A, B, C, D. What is the probability of getting an A?

P(A) = # of preferred outcomes (1)

       # of all possible outcomes (4)

So the P(A)= 1/4 =.25;
Similar as before, but now the A takes out more, so in reality we have A, A, C, D.

P(A) = # of preferred outcomes (2)
       __________________________________
       # of all possible outcomes (4)

So the P(A)= 2/4 =.50;

The probability of 2 independent events happening at the same time is their product.

So to get A in one circle and C in another circle: P(A) * P(C) = .5 * .25 = .125;

Sometimes we cannot calculate theoretical (above) probabilities and have to go with the experimental ones. In this case, we can calculate the probabilityof A with:
P(A) = # of trials where we got what we wanted
       __________________________________
       # of trials we did


If we have the experimental probability, we can calculate the expected outcome with n number of trials as:
Expected Outcome = E(A) = P(A) * n;
