The table shows the possible outcomes of rolling 2 dice. There are a total of 36 cells. Some numbers appear only in one cell, some in multiple cells. The probability of each outcome is:
Number of cells containing the number / 36;

Using the expected numerical outcome formula E(n) = P(2) * 2 + P(3) * 3 ... + P(12) * 12 = 7, which is the one that shows up in the most cells.
The number of times each number appears is the Frequency.
The collection of the frequencies and the probability associated to the frequencies is the Probability Frequency Distribution. These can be shown in different types of graphs.
An event is a certain outcome. The complement of the event is everything the event is not. If the event we want is getting a 6 on the roll of the dice, its complement is getting anything else that is not a 6.
The sume of all probabilities should add up to 1 (100%). If it is greater than 1, we have that some of the outcomes can happen at the same time. If it is less than 1, we have missed something.
If the event is indicated with A, its complement is A', and A'' = A. Also,
P(A) = 1-P(A')

Permutation: Number of ways in which we can arrange a set of elements.
We have 3 jumpers, whoever jumps the farthest wins. There are 3 positions. Only one jumper can be in position #1. Only one can be in position #2 and only one can be in position #3. The number of permutations we can have is:
P(n)= n*(n-1)*(n-2)...*1 = n! (factorial)

0! = 1; Negative numbers do not have a factorial.
(n+1)! = n! * (n+1)
(n+k)! = n! * (n+1)*(n+2)*...(n+k)
(n-k)! = n! / ( (n-k+1)(n-k+2)...(n-k+k) )
if n> k ==> n! / k! = (k+1)(k+2)...n

So in the jumpers example, we have n! = 3! = 3*2*1 = 6 possible ways that this could end up if our jumpers are A, B, C:
